# Token Monitor

AIサービス（ClaudeCode / Codex）のトークン使用状況をデスクトップの極小ウィンドウで常時監視するツールです。

- 各サービスの公式ダッシュボードと同じ数値（セッション/週次/月間の使用率・リセット時刻・クレジット残高）を取得します。
- ブラウザを開かずに確認できます。
- 閾値超え・リセット直前のデスクトップ通知に対応しています。

## 検証状況（重要）

- **ClaudeCode**: Pro プランで動作確認済みです。
- **Codex**: Free プランでのみ動作確認しています。有料プラン（Plus/Pro等）で表示項目が変わる可能性がありますが未検証です。挙動に差異があれば Issue 等でお知らせください。

## 使い方

1. ビルド:
   ```
   cargo build --release
   ```
2. `config.example.toml` を `config.toml` にコピーします（このファイルは Git 管理対象外です）。
3. 実行ファイル（`token-monitor`）を起動し、⚙設定 からログイン情報を貼り付けます。手順はアプリ内のガイド（❓ 取得方法）を参照してください。

## セキュリティ上の注意

- `config.toml` にはログインセッション情報（Cookie 相当）が平文で保存されます。他人と共有したり公開リポジトリにコミットしないでください（`.gitignore` 済み）。
- 取得する情報はご自身のアカウントの使用状況のみです。

## 対応サービス

- ClaudeCode（Pro）
- Codex（Free）
- Antigravity は現状 API 未確認のため未対応です。
